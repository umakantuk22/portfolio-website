# Engineering Hub Portfolio

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## 🚀 Overview
A high-performance, professional engineering portfolio designed to showcase software engineering progress, technical stack evolution, and system design capability.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
* **Styling:** CSS Grid/Flexbox, FontAwesome
* **Architecture:** Component-based structure with dynamic rendering

## 🏗 Key Features
- **Dynamic Skill Radar:** Visualizes core foundations, production-ready skills, and AI integration.
- **Engineering Journal:** Integrated technical log for documenting architectural trade-offs.
- **Responsive Design:** Fully mobile-optimized layout using a mobile-first approach.
- **Smooth UX:** Entrance animations for project cards to improve user engagement.
